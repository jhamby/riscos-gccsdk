Autobuilt Applications and Libraries

Part of www.riscos.info

These pages contain applications and libraries that have been built with the GCCSDK autobuilder.

The recommended way to access the packages on these pages is using the package manager from the RISC OS Packaging Project

You will need to add this sites package index to the !RiscPkg sources file (Choices:RiscPkg.Sources) by adding the following line:

  pkg http://www.riscos.info/packages/pkg/autobuilt

You can browse the indexes to see what is currently available or to directly download the packaged zip file to install manually.

Finding a package

An application or library can be easily found using one of the following:

  • Alphabetical Index
  • Section/Category Index

Source code

Many of the packages on this site are released under licences that require the source code to be provided. Source code for these packages can be downloaded from http://www.riscos.info/packages/src/

Warning

The applications and libraries provided on these pages are intended to be rebuilt by the autobuilder regularly. This means that it is possible that they may not run if a change in upstream sources or the GCCSDK compiler introduces a bug.
